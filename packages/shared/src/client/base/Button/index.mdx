## Usuage
```tsx
<>
<Button as={Fragment}>
    {({ active, disabled, focus, hover }) => {
    return <button>button</button>;
    }}
</Button>
<style>
    // data-hover, data-disabled, ...
    {`
    button[data-hover] {
    background-color: black;
    color: white;
    }
`}
</style>
</>
```

## Detail
<div style={{ display: "flex", flexDirection: "column", gap: "50px" }}>
  <ul>
    <li>
      <strong>Render Props</strong>를 통해 다음 상태 값을 전달받을 수 있습니다:
      <code>active</code>, <code>disabled</code>, <code>focus</code>, <code>hover</code>
    </li>
    <li>
      상태에 따라 자동으로 부여되는 <code>data-*</code> 속성을 기반으로 
      <strong>CSS 스타일링</strong>이 가능합니다.
    </li>
    <li>
      <strong>Focus 상태</strong>는 Tab 키로 이동할 때만 활성화됩니다.
      마우스 클릭 시에는 focus 상태가 적용되지 않으며,
      <code>focus === true</code> 및 <code>data-focus</code> 속성은 Tab 키로 포커스될 때만 설정됩니다.
    </li>
  </ul>
</div>
