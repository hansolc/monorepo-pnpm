## Overview


<div style={{ display: "flex", gap: "32px", justifyContent: "space-between" }}>
  <div style={{ flex: 1 }}>
    <h3>고려한 것</h3>
    <ul>
      <li>Default type</li>
      <li>Leading Icon 포함</li>
      <li><code>elevated</code>, <code>filled</code>, <code>outlined</code>, <code>tonal</code>, <code>text</code> 스타일 지원 </li>
      <li><code>xs</code>, <code>sm</code>, <code>md</code>, <code>lg</code>, <code>xl</code> 크기 지원 </li>
      <li><code>round</code>, <code>square</code> 지원</li>
      <li>각 hover, focus, disabled, active 마다 스타일 대응</li>
    </ul>
  </div>
  <div style={{ flex: 1 }}>
    <h3>고려하지 않은 것</h3>
    <ul>
      <li>Toggle type: 버튼 누른상태와 누르지 않은 상태 구분하는 버튼(ex) 재생버튼. 재생 누르기전 상태: play, 누른 후: pause)</li>
      <li>Shape morph: 각 shape 마다 enable, hover, pressed 상태에서 스타일 애니매이션 변화</li>
    </ul>
  </div>
</div>
