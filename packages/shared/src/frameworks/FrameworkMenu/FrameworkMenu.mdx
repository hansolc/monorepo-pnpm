# FrameworkMenu

<br/>
<br/>

## Overview
- 사용자가 목록에서 하나의 옵션을 선택할 수 있도록 하는 UI 컴포넌트입니다.
- Material Design 3의 `Menu` 디자인 가이드를 반영해 구현되었습니다.
- `Dropdown` 컴포넌트를 기반으로 구성되어 있으며, `trigger`, `options`, `onChange` 등의 props를 통해 유연하게 제어할 수 있습니다.
- 트리거 요소는 `<Dropdown.Trigger>`를 통해 렌더링되며, 원하는 형태의 ReactElement를 `as` prop으로 전달해 커스터마이징할 수 있습니다.

## Usuage

```tsx
<Menu
    trigger={
    <fieldset className={fieldSet({ type: "outlined" })}>
        <legend className={legendStyle({ floated: true })}>{label}</legend>
        <Flex justify="between">
        {String(selected ?? "선택해주세요.")}
        <MdArrowDropDown size="24" />
        </Flex>
    </fieldset>
    }
    options={options}
    onChange={change}
/>
```

## Properties

### Options

- `Array<string>` 형태의 옵션 리스트입니다.
- 각 항목은 내부적으로 `<Dropdown.Item>`으로 렌더링됩니다.
---

### Trigger
- 메뉴를 열고 닫는 트리거 요소입니다.
- `ReactElement` 타입이며, `<Dropdown.Trigger as={...}>` 내부에서 클릭 이벤트가 자동으로 바인딩됩니다.
- 버튼, `fieldset`, 기타 HTML 요소 등 원하는 형태로 구성할 수 있습니다.
---

### onChange
- 옵션이 선택될 때 호출되는 콜백 함수입니다.
- 선택된 값이 인자로 전달되며, 외부 상태를 갱신하는 데 사용할 수 있습니다.
---

### selected
- 현재 선택된 옵션의 값입니다.
- `value`와 동기화되어 있으며, 외부 상태에서 제어하거나 초기값 설정에 사용할 수 있습니다.
---

### valueAs
- 사용자에게 표시되는 텍스트와 실제 선택 값이 다를 경우 사용하는 포맷터 함수입니다.
- `(value: string) => string` 형태로 전달하며, 각 옵션을 어떻게 표시할지 커스터마이징할 수 있습니다.
