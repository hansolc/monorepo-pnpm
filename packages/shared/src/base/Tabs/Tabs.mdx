# Tabs 문서

`Tabs`는 Compound Pattern 기반의 컴포넌트로, `TabList`, `Label`, `TabPanel`, `Tab` 하위 컴포넌트를 조합하여 다양한 입력 UI를 만들 수 있습니다.

---

## Usage
```tsx
const Tabs = ({ options, selected, onSelect }: Props) => {
  return (
    <TabProvider selected={selected} onSelect={onSelect} ariaLabel="example">
      <TabLabel label="category" />
      <TabList>
        {options.map((option) => (
          <Tab value={option} as={"button"} />
        ))}
      </TabList>
    </TabProvider>
  );
};

```
